Square grid, simple enough

Triangular grid, find intersection of line1 and line2 line2 is horizontal: y = n2 * y_factor + origin.y y_factor is the height of the triangle sqrt(3)/2 * size. line1 has a 30deg slope, and x offset of n1*size + origin.x y = sqrt(3) * ( x + n1*size + origin.x ) They intersect in the point where y is constrained by line2, x = size/2 * n2 + size * n1

if you think of p as (x,y)+offset, you can get the expressions used below to get (x,y) from (p.x,p.y). The rounding p/size removes the offset as |offset.x| < size
